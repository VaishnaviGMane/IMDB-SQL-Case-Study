#  IMDB Data Analysis Using SQL

##  Executive Summary
This project presents an end-to-end SQL case study performed on the IMDB dataset.  
The objective was to analyze movie, rating, and director data to generate meaningful business insights using advanced SQL techniques.

The case study simulates real-world business analysis scenarios commonly asked in Data Analyst interviews and assessments.

---

##  Problem Statement
A media analytics company wants to analyze IMDB movie data to:

- Identify top-performing movies
- Analyze director performance
- Evaluate rating trends
- Perform genre-based insights
- Rank movies using analytical SQL functions

---

##  Technical Stack
- MySQL
- SQL (Advanced Queries)
- Relational Database Concepts
- GitHub for Version Control

---

##  Analytical Approach

### Data Exploration
- Understanding table relationships
- Identifying primary and foreign keys

### Data Analysis
- Aggregation using GROUP BY
- Filtering using HAVING
- Multi-table joins
- Subqueries for nested analysis
- Window functions (RANK, DENSE_RANK)

---

##  Key Insights Generated

- Identified highest-rated movies
- Ranked movies based on ratings
- Found directors with maximum movie contributions
- Calculated average ratings across genres
- Extracted Nth highest rated movies
- Performed trend-based analysis using sorting & filtering
